# Assets

This folder will hold RINGANA's brand assets — logos, certification badges, photography.

## What's here

Currently empty.

## What should be here (please supply)

- `logo.svg` — the RINGANA wordmark (and a white-on-dark variant)
- `wordmark-fresh.svg` — the FRESH sub-brand wordmark
- Certification marks (raster or vector):
  - COSMOS Organic
  - Vegan Society
  - Climate Neutral / CO₂-neutral
  - Made in Austria
- 3–5 hero photography references (.jpg) — product on flat-color ground, model candids, botanical still life
- Packaging renders (.png with transparency) for FRESH, CAPS, PACKS, DRINKS lines

Until those land, the UI kit renders the wordmark as live text using `.r-wordmark` (defined in `colors_and_type.css`) and uses neutral image placeholders sized to the brand's typical aspect ratios.

## Icons

Icons come from **[Lucide](https://lucide.dev/)** via CDN — `https://unpkg.com/lucide@latest`. This is a substitution; RINGANA's actual icon set is not in the source materials. The match is close (1.5 px stroke, round caps, geometric proportions). Swap to the real icons when available.
