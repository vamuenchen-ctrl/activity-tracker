# Re-bundling `index.html`

The single-file `index.html` is generated by concatenating all `.jsx` and `app.css` source files into one document. To re-bundle after edits, run this script in the project (e.g. via the AI's run_script tool):

```js
const css1 = await readFile('colors_and_type.css');
const css2 = await readFile('aktivitaetsmesser/app.css');
const files = {
  ios:  await readFile('aktivitaetsmesser/ios-frame.jsx'),
  data: await readFile('aktivitaetsmesser/data.jsx'),
  ui:   await readFile('aktivitaetsmesser/ui.jsx'),
  onb:  await readFile('aktivitaetsmesser/onboarding.jsx'),
  home: await readFile('aktivitaetsmesser/home.jsx'),
  cal:  await readFile('aktivitaetsmesser/calendar.jsx'),
  set:  await readFile('aktivitaetsmesser/settings.jsx'),
  app:  await readFile('aktivitaetsmesser/app.jsx'),
};
const fontImport = (css1.match(/@import url\([^)]+\);/) || [''])[0];
const cssCore = css1.replace(/@import url\([^)]+\);/, '');
const allCss = cssCore + '\n\n/* ===== app.css ===== */\n' + css2;

const tag = (label, src) => `<script type="text/babel" data-presets="react">\n/* ===== ${label} ===== */\n${src}\n</script>`;

const html = `<!doctype html>
<html lang="de">
<head>
<meta charset="utf-8" />
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no" />
<meta name="theme-color" content="#677E3D" />
<title>AKTIVITÄTSMESSER</title>
${fontImport ? `<style>${fontImport}</style>` : ''}
<link rel="manifest" href="manifest.webmanifest" />
<style>${allCss}</style>
<script src="https://unpkg.com/react@18.3.1/umd/react.development.js" integrity="sha384-hD6/rw4ppMLGNu3tX5cjIb+uRZ7UkRJ6BPkLpg4hAu/6onKUg4lLsHAs9EBPT82L" crossorigin="anonymous"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.development.js" integrity="sha384-u6aeetuaXnQ38mYT8rp6sbXaQe3NL9t+IBXmnYxwkUI2Hw4bsp2Wvmx4yRQF1uAm" crossorigin="anonymous"></script>
<script src="https://unpkg.com/@babel/standalone@7.29.0/babel.min.js" integrity="sha384-m08KidiNqLdpJqLq95G/LEi8Qvjl/xUYll3QILypMoQ65QorJ9Lvtp2RXYGBFj1y" crossorigin="anonymous"></script>
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.min.js"></script>
</head>
<body>
<div id="root"></div>
${tag('ios-frame.jsx', files.ios)}
${tag('data.jsx',      files.data)}
${tag('ui.jsx',        files.ui)}
${tag('onboarding.jsx',files.onb)}
${tag('home.jsx',      files.home)}
${tag('calendar.jsx',  files.cal)}
${tag('settings.jsx',  files.set)}
${tag('app.jsx',       files.app)}
</body>
</html>`;

await saveFile('aktivitaetsmesser/index.html', html);
```

## Why bundled at all?

During development, the project's static-file serving started returning 401s for project-relative `<link>` and `<script src>` paths — `index.html` would load but every CSS/JSX dependency would fail. Inlining everything into a single document sidesteps that entirely. CDN-loaded scripts (React, Babel, Lucide, Google Fonts) work fine.
